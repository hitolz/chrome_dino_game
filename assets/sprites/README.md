# 游戏素材说明

请将以下图片文件放入此目录：

## 必需的图片文件：

1. **dino.png** (恐龙)
   - 尺寸：40x40 像素
   - 格式：PNG（带透明背景）
   - 描述：小恐龙角色，建议绿色或灰色

2. **cactus.png** (仙人掌)
   - 尺寸：30x50 像素  
   - 格式：PNG（带透明背景）
   - 描述：仙人掌障碍物，建议深绿色

3. **ground.png** (地面)
   - 尺寸：100x20 像素
   - 格式：PNG（带透明背景）
   - 描述：地面纹理，建议棕色或灰色

## 获取素材的方法：

1. **自己绘制**：使用 Photoshop、GIMP、或在线工具如 Canva
2. **免费素材网站**：
   - OpenGameArt.org
   - Itch.io (免费游戏素材)
   - Pixabay
3. **简单几何图形**：可以先用纯色方块代替，后续再美化

## 临时解决方案：
如果暂时没有图片，游戏会使用纯色方块作为占位符。
